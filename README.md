# Snehal Printers - Modern Web Application

A cutting-edge, modernized version of the Snehal Printers website built with React, Vite, Tailwind CSS, and Framer Motion.

## 🚀 Tech Stack

- **Frontend**: React 18 + Vite
- **Styling**: Tailwind CSS 3.3
- **Animations**: Framer Motion 10.16
- **Icons**: Font Awesome (via CDN)
- **Font**: Montserrat (Google Fonts)

## 📦 Prerequisites

Before running this project, ensure you have the following installed:

- **Node.js** (v16 or higher) - [Download here](https://nodejs.org/)
- **npm** (comes with Node.js) or **yarn**

## 🛠️ Installation & Setup

1. **Clone or download this project**

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000`

## ✨ Features

### 🎨 Modern Design
- **Dark/Light Mode**: Toggle between themes with smooth transitions
- **Glassmorphism**: Modern glass-effect UI elements
- **Gradient Text**: Eye-catching gradient typography
- **Responsive Design**: Fully responsive across all devices

### ⚡ Performance
- **Vite**: Lightning-fast HMR (Hot Module Replacement)
- **Optimized Images**: All original images preserved
- **Smooth Animations**: 60fps animations with Framer Motion

### 🎯 Components
- **Header**: Fixed navigation with scroll effects
- **Hero**: Animated hero with gradient orbs
- **About**: Company info with floating badges
- **Services**: Service cards with hover effects
- **Portfolio**: Product gallery with lightbox modal
- **Testimonials**: Client reviews with quote icons
- **Clients**: Infinite scrolling logo carousel
- **Contact**: Modern form with validation
- **Footer**: Multi-column footer with links
- **BackToTop**: Animated scroll-to-top button
